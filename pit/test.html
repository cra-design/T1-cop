Choose a filing method – File your GST/HST return

Learn about the different ways you can file a return, and choose the one that works best for you.

On this page

  • Mandatory electronic filing
  • Filing method options
  • Compare what you can do with each method
  • Access code requirements

Mandatory electronic filing

If you are a GST/HST registrant with a reporting period that begins in 2024 or later, you must file your returns electronically (except for charities and selected listed financial institutions). If you don't, we may charge you a penalty.

Learn more: Mandatory electronic filing requirements and exceptions

Filing method options

 In your online CRA account

File your GST/HST return directly in your CRA account (My Business Account or Represent a Client) using NETFILE.

You can:

  • Meet electronic filing requirements
  • File without an access code
  • Include schedules
  • File certain rebates
  • File elections
  • Make a payment
  • Get a confirmation number right away
  • File a joint GST/HST and QST return (RC7200)

You cannot:

  • File using this method unless you have an online CRA account

Sign in to your CRA account

Learn how to file: Using your CRA account

 Using the online form with an access code

If you do not have a CRA account (My Business Account or Represent a Client) you can file your GST/HST return using the NETFILE form online.

This filing method requires a GST/HST access code.

What you can and cannot do using this method

You can:

  • Meet electronic filing requirements
  • File without an online CRA account
  • Include schedules
  • File certain rebates
  • Get a confirmation number right away
  • File a joint GST/HST and QST return (RC7200)

You cannot:

  • File without an access code
  • File elections
  • Make a payment

File your return using the NETFILE form

Learn how to file: Using the NETFILE form with an access code

 Through a financial institution

You can file your return and remit your GST/HST online through a participating Canadian financial institution using Electronic Data Interchange.

What you can and cannot do using this method

You can:

  • Meet electronic filing requirements
  • File without an access code
  • File without an online CRA account
  • Include schedules B and C only
  • Make a payment

You cannot:

  • Include schedules other than B and C
  • File rebates
  • File elections
  • File a joint GST/HST and QST return (RC7200)
  • Get a confirmation number right away

Learn how to file: Through a financial institution

 Using third-party accounting software

You can prepare your GST/HST return using third-party accounting software and upload it directly to the CRA through GST/HST Internet File Transfer.

This filing method requires a GST/HST access code.

What you can and cannot do using this method

You can:

  • Meet electronic filing requirements
  • File without an online CRA account
  • Get a confirmation number right away

You cannot:

  • File without an access code
  • Include schedules
  • File rebates
  • File elections
  • Make a payment
  • File a joint GST/HST and QST return (RC7200)

Upload your return created using third-party accounting software

Learn how to file: Using third-party accounting software

 By phone

You can file your GST/HST return by phone using CRA's TELEFILE service.

This filing method requires a GST/HST access code.

What you can and cannot do using this method

You can:

  • Meet electronic filing requirements
  • File without an online CRA account
  • Get a confirmation number right away

You cannot:

  • File without an access code
  • Include schedules
  • File rebates
  • File elections
  • Make a payment
  • File a joint GST/HST and QST return (RC7200)

Learn how to file: Using TELEFILE

 By mail

You can file by mail by sending your paper return (GST34-2, GST62 or RC7200) to the address on the return.

Possible penalty if you file by mail

You may be charged a penalty if you file by mail, unless you are a charity, a selected listed financial institution, or meet another exception.

Read more about mandatory electronic filing and exceptions

What you can and cannot do using this method

You can:

  • File without an online CRA account
  • File without an access code
  • Include schedules
  • File rebates
  • File elections
  • Make a payment (cheque only)
  • File a joint GST/HST and QST return (RC7200)

You cannot:

  • Meet electronic filing requirements
  • Get a confirmation number right away

Learn how to file: By mail

Access code requirements

You will need a GST/HST access code if you file your GST/HST return using:

  • The NETFILE form online (outside of your CRA account)
  • Third-party accounting software (Internet File Transfer)
  • TELEFILE
